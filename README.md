# 🏆 社团管理系统 (Club Management System)

一个基于 **Spring Boot 3** 和 **Vue 3** 开发的高颜值、功能完备的社团管理平台。

## ✨ 功能亮点
- 📊 **数据看板**：实时汇总社团成员总数与活动开展情况。
- 👥 **成员管理**：支持成员信息的动态展示与一键踢出操作。
- 📅 **活动管理**：完整的 CRUD（增删改查）流程，支持关键词模糊搜索。
- 🛡️ **安全加固**：集成 Spring Security，保障后端接口安全。

## 🛠️ 技术栈
- **后端**: Java 17, Spring Boot, MyBatis-Plus, Spring Security, MySQL
- **前端**: Vue 3, Vite, Axios, Element Plus

## 🚀 快速启动
1. 导入 `sql/` 目录下的数据库脚本。
2. 在后端 `application.yml` 中配置你的 MySQL 连接。
3. 运行后端 `ClubApplication`，前端执行 `npm run dev`。
